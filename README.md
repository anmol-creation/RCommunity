# R Community

**R Community** is a mobile-first social media platform built exclusively for delivery riders and gig workers. It provides a safe, trusted, and verified space for workers to connect, share experiences, and support each other without employer surveillance.

## Project Documentation

*   [**Product Design & Architecture**](./DESIGN.md): Detailed overview of the product vision, features, system architecture, and tech stack.

## Key Features

*   **Rider-Only Access**: Full features are unlocked only after verifying rider status.
*   **Safe & Private**: No Aadhaar/Government ID required. Privacy-first design.
*   **Community Focused**: A space to discuss work issues, daily challenges, and solidarity.

## Tech Stack (Proposed)

*   **Mobile**: React Native (TypeScript)
*   **Backend**: Node.js (TypeScript)
*   **Database**: PostgreSQL
